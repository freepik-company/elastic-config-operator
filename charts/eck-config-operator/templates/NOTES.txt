üöÄ Quick Start:

1. Verify the operator is running:
   kubectl get deployment -n {{ .Release.Namespace }} {{ include "eck-config-operator.fullname" . }}

2. Check operator logs:
   kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/name={{ include "eck-config-operator.name" . }} -f

3. Create your first IndexLifecyclePolicy:
   kubectl apply -f - <<EOF
   apiVersion: eck-config-operator.freepik.com/v1alpha1
   kind: IndexLifecyclePolicy
   metadata:
     name: my-ilm-policy
   spec:
     syncInterval: "30s"
     resourceSelector:
       name: elasticsearch
       namespace: default
     resources:
       30d-retention:
         policy:
           phases:
             hot:
               min_age: "0ms"
               actions:
                 rollover:
                   max_age: "7d"
             delete:
               min_age: "30d"
               actions:
                 delete: {}
   EOF

4. Check the status:
   kubectl get indexlifecyclepolicies
   kubectl describe indexlifecyclepolicy my-ilm-policy
